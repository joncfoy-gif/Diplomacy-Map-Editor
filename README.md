# Diplomacy Map Editor

A browser based map editor for Diplomacy style maps.

## Features

- Draw territories as polygons by clicking on the canvas  
- Name each territory  
- Mark territories as supply centers  
- Create adjacency connections between territories  
- Export and import maps as JSON

## Usage

1. Open the app in your browser.
2. Click **Add territory**, then click points on the canvas to define the shape.
3. Click **Finish territory** and give it a name.
4. Use **Select / inspect** to click inside a territory and select it.
5. Use **Toggle supply center** to mark it as a supply center.
6. Use **Add connection** to click two territories and connect them.
7. Use **Export map** to save your work as JSON, and **Import map** to load it again.
